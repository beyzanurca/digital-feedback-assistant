# Digital Feedback Assistant

This project is a prototype application developed to provide AI-based digital feedback in educational settings.

## Purpose
The purpose of this application is to analyze students' written responses and provide constructive, motivating, and learning-oriented feedback using artificial intelligence.

## Features
- Analyzes student responses
- Provides instant formative feedback
- Supports learning-oriented assessment
- Designed for educational communication

## Technologies Used
- Google AI Studio
- Gemini AI Model
- Prompt-based AI interaction

## Academic Context
This prototype was developed within the scope of an academic study on digital feedback and artificial intelligence in education.
